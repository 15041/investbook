Бэкап и восстановление данных

На этой странице вы можете:

  1. Выполнить экспорт данных в формате "Portfolio Open Format" (получить бэкап файл).
  2. Восстановить данные из ранее подготовленного бэкап файла.
  3. Импортировать в Investbook данные из другого приложения учета инвестиций, который поддерживает экспорт данных в "Portfolio Open Format".
  4. Экспортировать данные в формате "Portfolio Open Format" для переноса в другое приложение учета инвестиций.

Восстановление данных может пригодиться, если вы вносили данные через Формы (Investbook Forms) и хотите защитить данные от сбоев вашего компьютера.

Если вы загружали данные только из отчетов брокеров, восстановление данных можно выполнить еще одним способом - повторно загрузив отчеты брокера. Они хранятся в домашней директории пользователя в папке "investbook\report-backups".

Вернуться на главную страницу



1. Для того чтобы получить бэкап файл сейчас, нажмите на эту ссылку.

2. Для того чтобы восстановить данные, загрузите ранее полученный бэкап файл.

Обратите внимание, что восстановление добавляет данные к уже имеющимся. Если вы хотите загрузить данные "на чистую", существующие данные можно удалить через действие "Удалить счет" на странице Форм.
